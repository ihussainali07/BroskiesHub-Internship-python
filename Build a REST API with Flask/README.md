# Flask User API 🧑‍💻

A simple RESTful API built with Flask that performs **CRUD operations** (Create, Read, Update, Delete) for managing users using an in-memory dictionary.

## 🚀 Features

- Create a new user
- Retrieve all users
- Retrieve a single user by ID
- Update an existing user
- Delete a user

## 📦 Requirements

- Python 3.x
- Flask
